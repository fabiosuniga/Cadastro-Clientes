# Sistema de Cadastro de Clientes

Uma aplicação CRUD simples construída em Python usando a interface gráfica **Tkinter**. 
O sistema permite cadastrar clientes em um banco de dados **SQLite** e exportar os registros diretamente para uma planilha em **Excel** utilizando **Pandas**.

## Como executar
1. Instale o Pandas e o openpyxl (necessário para exportar o excel):
`pip install pandas openpyxl`
2. Execute o arquivo principal:
`python app.py`
